Why This Site Exists

GitHub can host the website, not the Python server

GitHub Pages is a static hosting platform. That means HTML, CSS, and JavaScript are fine, but the live local dashboard from ai-sales-agent cannot run on GitHub itself because it needs a Python process and a writable SQLite database.

This page gives the project a clean hosted presence on your GitHub account. The operational dashboard stays local, where it can access your database and workflow tools safely.

Local dashboard: http://127.0.0.1:8788
Publish Steps

Enable GitHub Pages

1. Push these files to `main`

The repository needs index.html at the root so GitHub Pages has a site entrypoint.

2. Open Repository Settings

Go to Settings -> Pages in EmadAmeri/ai-agents.

3. Choose branch deploy

Select Deploy from a branch, choose main, and folder /(root).

4. Visit the site

After the first build completes, open https://emadameri.github.io/ai-agents/.
